# 🍅 Pomodoro Timer с задачами

Графический таймер Pomodoro на Python с использованием `tkinter`, включающий управление задачами, статистику и автоматические перерывы.

---

## 📋 Описание

Приложение реализует технику Pomodoro для повышения продуктивности:
- **25 минут** — рабочая сессия (Pomodoro)
- **5 минут** — короткий перерыв
- **15 минут** — длинный перерыв после каждых 4 Pomodoro
- Управление списком задач с отслеживанием прогресса
- Статистика выполненных сессий
- Автоматическое сохранение всех данных

---

## 🚀 Запуск

```bash
python pomodoro.py
```

**Требования:** Python 3.7+

**Зависимости:** только стандартная библиотека Python (`tkinter`, `json`, `threading`, `datetime`, `dataclasses`, `enum`)

---

## 🎮 Инструкция по использованию

### 1. Добавление задач
- Введите название задачи в поле ввода
- Нажмите **"➕ Добавить"** или клавишу `Enter`
- Задача появится в списке со статусом ⏳ В ожидании

### 2. Запуск Pomodoro
1. Выберите задачу в списке и нажмите **"▶️ Выбрать"**
2. Нажмите **"▶️ Старт"** для запуска таймера
3. Работайте 25 минут — таймер отсчитывает время
4. По завершении предложится перерыв

### 3. Управление задачами
| Кнопка | Действие |
|--------|----------|
| **▶️ Выбрать** | Назначить задачу для текущего Pomodoro |
| **✅ Завершить** | Отметить задачу как выполненную |
| **🗑️ Удалить** | Удалить задачу из списка |

### 4. Управление таймером
| Кнопка | Действие |
|--------|----------|
| **▶️ Старт** | Запустить таймер (требуется выбранная задача) |
| **⏸️ Пауза** | Приостановить таймер |
| **⏹️ Сброс** | Сбросить таймер на 25:00 |

## 🛠️ Архитектура

### Классы данных
```
TaskStatus (Enum)
├── PENDING     — ⏳ В ожидании
├── IN_PROGRESS — ▶️ В работе
└── COMPLETED   — ✅ Завершена

Task (dataclass)
├── id, name, status, created_at, pomodoros

Session (dataclass)
├── task_id, task_name, start_time, duration, completed
```

### Основной класс
```
PomodoroGUI
├── _create_styles()        # Цветовая схема
├── _create_widgets()       # Создание интерфейса
│   ├── Таймер (25:00)
│   ├── Прогресс-бар
│   ├── Кнопки управления
│   ├── Управление задачами
│   └── Статистика
├── _add_task()             # Добавление задачи
├── _select_task()          # Выбор активной задачи
├── _complete_task()        # Завершение задачи
├── _delete_task()          # Удаление задачи
├── _start_timer()          # Запуск Pomodoro
├── _timer_loop()           # Цикл таймера (поток)
├── _timer_complete()       # Завершение сессии
├── _offer_break()          # Предложение перерыва
├── _start_break()          # Запуск перерыва
├── _break_loop()           # Цикл перерыва
├── _break_complete()       # Завершение перерыва
├── _pause_timer()          # Пауза
├── _reset_timer()          # Сброс
├── _update_stats()         # Обновление статистики
├── _save_data()            # Сохранение в JSON
├── _load_data()            # Загрузка из JSON
└── _save_and_exit()        # Сохранение и выход
```

## ⚙️ Технические детали

| Параметр | Значение |
|----------|----------|
| Размер окна | 500×700 пикселей |
| Цветовая схема | Тёмная (фон `#2c3e50`, акцент `#e74c3c`) |
| Потоки | Отдельный поток для таймера (`threading.Thread`) |
| Обновление UI | Через `root.after()` для потокобезопасности |
| Формат данных | JSON с задачами, сессиями и счётчиком ID |
| Длительность Pomodoro | 25 минут |
| Короткий перерыв | 5 минут |
| Длинный перерыв | 15 минут (каждые 4 Pomodoro) |
